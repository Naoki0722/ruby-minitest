# Ruby-minitest

## 概要
純RubyとMinitestを使用してテストを実行するための環境。

## インストール

```bash
bundle install
```

## 使用方法
コードはlibディレクトリの中に格納。



## テスト
純正Minitestを使っています。

Minitestを使用してテストを実行するには、以下のコマンドを使用してください。

```bash
ruby test/{target_test_filename}.rb
```

テスト駆動開発の本を実際に手を動かして実装していくため。
